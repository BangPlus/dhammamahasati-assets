# Dhamma Mahasati Assets

Optimized public media assets for the Dhamma Mahasati website.

This repository stores lightweight, optimized image assets used for public Dhamma education, preservation, and non-commercial publication.

## Purpose

- Reduce dependency on external image hotlinks
- Improve long-term availability of Dhamma media
- Lower bandwidth usage by serving optimized WebP images
- Preserve source references for review and attribution

## Asset Policy

This repository is a reviewable mirror of selected public image assets referenced by Dhamma Mahasati pages.

Audio, video, PDF, CSS, JavaScript, and unknown files are not mirrored here.

Original sources are preserved in the audit and replacement plan files under `seo/`.

## Reports

- `seo/mirrored-assets-map.json`
- `seo/mirrored-assets-report.md`
- `seo/image-replacement-plan.json`
